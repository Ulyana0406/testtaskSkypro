Инструкция по запуску проекта
1. Запустите VS Code.
2. Склонируйте репозиторий на локальный компьютер.
3. Установите необходимые зависимости, используя команду npm install.
4. Перейдите в директорию проекта с помощью команды cd "<путь>".
5. Инициируйте запуск проекта командой npm start.
6. Откройте в браузере страницу проекта по адресу: http://localhost:3000.

Руководство пользователя
1. Поиск пользователей возможен по имени или логину.
2. Информация о количестве найденных пользователей отображается справа от кнопки “Найти”. Если поиск не дал результатов, вы увидите “Найдено: 0”.
3. Фильтрация доступна по числу репозиториев пользователя и может быть выполнена в порядке убывания или возрастания. Для работы с фильтром используйте кнопки “Фильтр” для его активации и “Очистить фильтр” для сброса.
4. Навигация по страницам и настройка количества пользователей на страницу доступны в нижней части экрана. При изменении настроек пагинации, отображение начнётся с первой страницы.
5. Клик по имени пользователя откроет окно с детальной информацией и ссылкой на его GitHub профиль.
6. Просмотр доступен для первых 1000 найденных пользователей.
